# Demo assets

Commit your teaching assets here so students get everything in one clone.

## Required files

```
environment/scene.blend     # full Blender environment
objects/tank/tank.blend     # one War Thunder tank (or .obj / .ply)
```

## Checklist before pushing

- [ ] Environment opens cleanly in Blender 3.x+
- [ ] Tank mesh has applied scale and sits at a sensible origin
- [ ] Combined scene looks correct when tank is at `[0, 0, 0]` (default)
- [ ] Total repo size is reasonable for your students (consider decimating high-poly tanks)
